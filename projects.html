Latest Projects

Iron Limitation in Acartia tonsa


My current project entails studying at what specific concentration of iron does the copepod Acartia tonsa experience limitation. This is acomplished through growing the diatom, Thalassiosira oceanica, under replete (100 nM Fe) and deplete (1 nM Fe) conditions and mixing the two cultures in different proportions to achieve different iron concentrations. I will be using egg production as a proxy for limitation in the copepods.

I plan to use this research as a requirement for my masters degree and will be working on publishing the results.


Emergence of asexual reproduction in Oenothera spp.


I am also involved in an upcomming project this fall which entails working bioinformatically with gene sequences to find an evolutionary threshold point that shows which genes have specifically evolved to cause asexual reproduction in Oenothera. Computational methods include working with data using command line tools on a Unix server and writing scripts in Python using the Biopython module.
